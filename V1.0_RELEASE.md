# KnowLeaf 知叶 v1.0 · 发布总结

> 2026-05-25 | 单文件 Web 应用 | 3215 行 | 纯前端

---

## 一句话

**拍照/打字就能提取知识点，自动生成思维导图，按遗忘曲线复习 —— 一个给大学生用的 AI 学习助手。**

---

## 核心功能

### 📖 学习板块
| 功能 | 说明 |
|---|---|
| 文本输入 | 输入学习笔记，AI 自动提取知识点（学科/知识点/总结） |
| 拍照上传 | 拍课件/笔记，OCR 识别 + 知识点提取（通义千问 VL） |
| 思维导图 | Markmap 实时渲染，支持多层级嵌套，可折叠展开 |
| 知识点编辑 | ✏️ 修改学科/知识点/总结，自动移动到新位置 |
| AI 深度分析 | 🤖 核心概念/关键要点/易错提醒/记忆技巧/关联知识/自测题 |
| 图片查看 | 知识树缩略图点击全屏放大 |

### 📝 复习板块
| 功能 | 说明 |
|---|---|
| 艾宾浩斯算法 | 间隔 1→2→4→7→15→30→90 天，自动计算下次复习时间 |
| 卡片式问答 | "知识点名称"是什么？→ 点击查看答案 → 答对/答错 |
| 进度追踪 | 待复习数量、已掌握数量、连续打卡天数 |
| 防刷机制 | 当天已完成复习后按钮变灰 |

### 👤 我的板块
| 功能 | 说明 |
|---|---|
| API 配置 | 多平台预设（通义千问/DeepSeek/SiliconFlow/Groq），自适应匹配模型 |
| 视觉模型检测 | 上传图片时自动检测当前模型是否支持视觉，不支持则拦截提示 |
| 数据统计 | 总条目/学科数/已掌握数/打卡天数 |
| 数据管理 | 导出 ZIP（JSON + 思维导图 PDF）、导入 JSON 恢复、清空数据 |
| 备份提醒 | 超过 7 天未导出时黄色警告条 |

## 技术栈

| 层 | 技术 |
|---|---|
| 前端框架 | 纯 HTML/CSS/JS，单文件 3215 行 |
| 思维导图 | Markmap (D3.js + markmap-autoloader 0.15) |
| 数据存储 | localStorage (JSON 文字) + IndexedDB (图片) |
| AI 接口 | OpenAI 兼容 API（通义千问 qwen-vl-max 为主） |
| 导出 | jsPDF 2.5.2 + JSZip 3.10.1（CDN） |
| 复习算法 | 艾宾浩斯遗忘曲线（自实现） |
| 图片处理 | Canvas 压缩、FileReader、Blob URL |

---

## 架构决策

- **单文件架构**：不拆分桌面/移动端，用响应式 CSS 适配（`@media min-width: 768px`）
- **图片分离存储**：文字存 localStorage，图片存 IndexedDB（解决 5MB 上限 + 清晰度问题）
- **AI 自适应匹配**：根据 API URL 自动识别平台 → 列出可用模型 → 标注视觉能力
- **知识粒度细化**：AI prompt 强制要求多条规则/步骤拆分为独立 topic + parentTopic 层级关联
